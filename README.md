
# Pizza Sales Data Analysis using SQL


############ Project Overview ############

This project analyze Pizza Hut sales data using MySQL.

The goal is to extract meaningful business insights such as revenue generation, best-selling pizzas, order trends, customer purchasing behavior, and category-wise performance using SQL queries.

############ Dataset ############

The project uses four tables:

- Orders: order_id, order_date, order_time
- Order Details: order_details_id, order_id, pizza_id, qunatity
- Pizzas: pizza_id, pizza_type_id, size, price
- Pizza Types: pizza_type_id, name, category, ingredients

############ SQL Concepts Used ############

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- INNER JOIN
- Subqueries
- Window Functions
- RANK()
- ROUND()
- Date & Time Functions

############ Business Questions Solved ############

✔ Total Orders

✔ Total Revenue

✔ Highest Priced Pizza

✔ Most Ordered Pizza Size

✔ Top 5 Pizza Types

✔ Category-wise Quantity Sold

✔ Orders by Hour

✔ Daily Average Orders

✔ Top Revenue Generating Pizza Types

✔ Top 3 Revenue Generating Pizzas in Each Category
